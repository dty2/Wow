# 设计

## 工作目录

```
--work-dir 指定工作目录$WORK_DIR, 默认在$XDG_RUNTIME_DIR
本地套接字监听在$WORK_DIR/wow/wow.sock
```

## IPC

### 请求

- refresh: `{ "action": "refresh" }`
- next: `{ "action": "next" }`
- prev: `{ "action": "prev" }`
- manual: `{ "action": "manual" }`
- test:

```
{
  "action": "test",
  "target": "[wallpaper-name]"
}
```

- list:

```
{
  "action": "list",
  "target": "[list-name]"
}
```

- quit: `{ "action": "quit" }`
- info: `{ "action": "info" }`

### 响应

`{ "result": "[result-info]" }`
