# 测试

```txt
.
├── cases  # <- 放置 ".test" 文件
│   ├── help.test
│   ├── list.test
│   ├── manual.test
│   ├── next.test
│   ├── previous.test
│   ├── refresh.test
│   ├── test.list
│   ├── unique-instance.test
│   └── version.test
├── CMakeLists.txt
├── fixtures
│   ├── config  # <- 模拟目录 $XDG_RUNTIME_DIR/wow
│   └── workdir  # <- 模拟目录 $XDG_CONFIG_HOME/wow
├── modify.sh
├── path.py.in
└── runtest.py  # <- 回归测试脚本, 运行 ".test" 文件
```

> [!NOTE]
> 添加新".test"文件之后, 请重新执行 `cmake --build build`

```bash
cd build

# 运行所有测试
ctest

# 单独运行测试
python3 ./test/runtest.py ../test/cases/[测试名].test
```
